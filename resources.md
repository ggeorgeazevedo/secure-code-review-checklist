# 📚 Resources & Further Reading

Curated references for going deeper on secure coding, code review, and AI/LLM security. Official OWASP material first, then free hands-on learning, then courses.

---

## 🦉 OWASP — core projects

| Project | What it's for |
|---|---|
| [OWASP Top 10:2025](https://owasp.org/Top10/2025/) | The baseline web app risk list this checklist maps to |
| [OWASP Top 10 for LLM Applications 2025](https://genai.owasp.org/llm-top-10/) | Risk list for LLM-powered apps (see the [AI/LLM checklist](ai-llm/owasp-llm-top-10.md)) |
| [OWASP Cheat Sheet Series](https://cheatsheetseries.owasp.org/) | Concise, actionable defense guidance per topic |
| [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/) | Application Security Verification Standard — requirements to verify against |
| [OWASP WSTG](https://owasp.org/www-project-web-security-testing-guide/) | Web Security Testing Guide — how to test each risk |
| [OWASP Proactive Controls](https://owasp.org/www-project-proactive-controls/) | Top security controls every developer should build in |
| [OWASP SAMM](https://owaspsamm.org/) | Software Assurance Maturity Model — measure & improve your program |
| [OWASP GenAI Security Project](https://genai.owasp.org/) | Hub for LLM/GenAI security guidance and checklists |

## 🧵 Threat modeling & design

- [OWASP Threat Modeling](https://owasp.org/www-community/Threat_Modeling) — process overview
- [OWASP Threat Dragon](https://owasp.org/www-project-threat-dragon/) — free threat modeling tool
- [OWASP pytm](https://owasp.org/www-project-pytm/) — threat modeling as code
- Methodologies referenced in the checklists: **STRIDE**, **PASTA**, **LINDDUN**

## 🔗 Supply chain & tooling

- [OWASP Dependency-Check](https://owasp.org/www-project-dependency-check/) — SCA for known-vulnerable deps
- [CycloneDX](https://cyclonedx.org/) — SBOM standard
- [Semgrep](https://semgrep.dev/) — fast, rule-based SAST across many languages
- Language-specific tools are listed in each checklist's **References** section

## 🎯 Hands-on practice (free)

- [PortSwigger Web Security Academy](https://portswigger.net/web-security) — free, excellent labs for web vulns
- [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) — intentionally vulnerable app to practice on
- [OWASP WebGoat](https://owasp.org/www-project-webgoat/) — guided vulnerable app with lessons
- [Google Gruyere](https://google-gruyere.appspot.com/) — classic web security codelab

## 🎥 Learning channels

- [OWASP Foundation on YouTube](https://www.youtube.com/@OWASPGLOBAL) — AppSec talks & conference recordings
- [PortSwigger Research](https://portswigger.net/research) — deep web security research
- Conference archives: **OWASP AppSec**, **Black Hat**, **DEF CON** (search their official channels)

## 🎓 Courses & structured paths

- [Microsoft Learn — Security](https://learn.microsoft.com/training/browse/?terms=security) — free modules on secure development & cloud security
- [AWS Skill Builder — Security](https://skillbuilder.aws/) · [Google Cloud Skills Boost](https://www.cloudskillsboost.google/) — cloud security fundamentals
- [TryHackMe](https://tryhackme.com/) · [Hack The Box Academy](https://academy.hackthebox.com/) — guided offensive/defensive paths
